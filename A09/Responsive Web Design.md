
# Définition

Le Responsive Web Design (RWD) consiste à créer des sites web qui s'adaptent automatiquement à la taille de l'écran. 

Principes essentiels :
- Eléments flexibles
- Media queries
- Mise en page fluide
- Images ajustées à la résolution
- Test sur différents écrans pour garantir un rendu correct.


# Processus de conception

## Mobile first

- Le site est au départ adapté aux petits écrans
- Ajout des styles pour tablette et desktop par après
- Propriété au contenu essentiel
- CSS : Styles de base, media queries en min-width

## Desktop first

- Le site est au départ adapté aux grands écrans
- Simplification par la suite pour tablette et mobile
- Priorité à un affichage complet
- CSS : Styles de base, media queries en max-width
